## Install dotfiles

```bash
git clone https://github.com/Sly-Harvey/.dotfiles.git ~/.dotfiles && cd ~/.dotfiles && ./install.sh
```
> [!Note]
> To install my arch linux packages run the install script in pkgs/
> ```bash
> ./pkgs/install.sh
> ```
